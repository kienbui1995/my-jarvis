# MY JARVIS — Edition Strategy

## Community Edition (Open Source, MIT)
**Target**: Cá nhân, developer, power user
**Giá trị**: Trợ lý AI cá nhân đầy đủ, không cắt xén

### Bao gồm TẤT CẢ:
- ✅ Full intelligence pipeline (router, planner, skills learning, memory)
- ✅ 8 channels (Zalo, Telegram, WhatsApp, Slack, Discord, Web, Mini App)
- ✅ All tools (research, finance, health, documents...)
- ✅ Memory system (consolidation, decay, knowledge graph, preferences)
- ✅ Proactive engine (triggers, morning briefing, reminders)
- ✅ MCP Gateway (connect external services)
- ✅ Voice (STT/TTS)
- ✅ Mobile app
- ✅ Self-hosted (Docker Compose)
- ✅ Single-user mode

## Enterprise Edition (Proprietary, Paid)
**Target**: Doanh nghiệp, team, tổ chức
**Giá trị**: Multi-tenant, compliance, admin control, white-label

### Enterprise-only features:
- 🏢 Multi-tenant / Team workspaces (shared calendar, tasks, expenses)
- 🏢 RBAC (role-based access control: admin, manager, member)
- 🏢 SSO (SAML 2.0, OIDC — Google Workspace, Azure AD)
- 🏢 Admin Dashboard (user management, usage analytics, cost control)
- 🏢 White-label API (businesses embed Jarvis engine)
- 🏢 Automation Builder (visual no-code workflow)
- 🏢 Agent-to-Agent protocol (inter-org communication)
- 🏢 Compliance (data retention policies, audit export, GDPR)
- 🏢 Skill Marketplace (enterprise approval workflow)
- 🏢 Priority support + SLA
- 🏢 On-premise deployment support
- 🏢 Custom branding
